# Land

Goal of this project is to create a simple server for uploading and retrieving DOF checkpoints.

## Getting Started

Install the dependencies:
```bash
$ pixi install
```

Run the server:
```bash
$ pixi run dev
```

Or, using docker
```
$ docker build -t land:local .

$ docker run -p 8000:8000 -it land:local
```
